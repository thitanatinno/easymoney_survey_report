# Assets Directory

## Required Assets

### Logo
Place the INNO logo file here as `inno-logo.png` for use in the cover sheet.

- File name: `inno-logo.png`
- Format: PNG
- Recommended size: 150x80 pixels or similar aspect ratio
- Location: `/assets/inno-logo.png`

If the logo file is not present, the cover sheet will still generate without it.
